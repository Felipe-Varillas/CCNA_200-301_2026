# CCNA_200-301_2026
## 🗂️ Estructura del Repositorio (Ruta de Aprendizaje)

El contenido está estructurado en 3 niveles de dificultad progresiva, alineados con los objetivos oficiales del examen CCNA 200-301:

### 🟢 Nivel 1: Fundamentos de Red
Aquí se encuentran las bases teóricas, el funcionamiento de los datos en la red y el direccionamiento IP.
* 📁 [**01_Modelos_OSI_TCP**](./Nivel_1_Fundamentos/01_Modelos_OSI_TCP): Análisis de protocolos, encapsulamiento y Wireshark.
* 📁 [**02_Capa_Fisica_Ethernet**](./Nivel_1_Fundamentos/02_Capa_Fisica_Ethernet): Cableado, topologías y configuración básica de switches/routers.
* 📁 [**03_Direccionamiento_IPv4_IPv6**](./Nivel_1_Fundamentos/03_Direccionamiento_IPv4_IPv6): Subnetting, VLSM y tipos de direcciones IPv6.

### 🟡 Nivel 2: Infraestructura y Conectividad
El núcleo del CCNA. Configuraciones prácticas de conmutación de capa 2 y enrutamiento de capa 3.
* 📁 [**04_Switching_VLAN_STP**](./Nivel_2_Infraestructura_Conectividad/04_Switching_VLAN_STP): Creación de VLANs, enlaces troncales (802.1Q), Spanning Tree Protocol y EtherChannel.
* 📁 [**05_Enrutamiento_OSPF_Estatico**](./Nivel_2_Infraestructura_Conectividad/05_Enrutamiento_OSPF_Estatico): Rutas estáticas, rutas por defecto y configuración de OSPFv2 de área única.
* 📁 [**06_Redes_Inalambricas_WLAN**](./Nivel_2_Infraestructura_Conectividad/06_Redes_Inalambricas_WLAN): Arquitecturas WLC, APs y seguridad WPA.

### 🔴 Nivel 3: Servicios, Seguridad y Automatización
Servicios auxiliares, protección de la infraestructura y la evolución hacia las redes definidas por software.
* 📁 [**07_Servicios_IP_NAT_DHCP**](./Nivel_3_Servicios_Seguridad_Automatizacion/07_Servicios_IP_NAT_DHCP): Traducción de direcciones (NAT), DHCP, DNS y NTP.
* 📁 [**08_Seguridad_ACLs**](./Nivel_3_Servicios_Seguridad_Automatizacion/08_Seguridad_ACLs): Listas de Control de Acceso (ACLs estándar y extendidas), Port Security y fundamentos AAA.
* 📁 [**09_Automatizacion_SDN**](./Nivel_3_Servicios_Seguridad_Automatizacion/09_Automatizacion_SDN): APIs REST, formatos JSON/XML, arquitecturas SDN (Cisco DNA Center) y herramientas de gestión de configuración.
